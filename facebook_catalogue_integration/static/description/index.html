Facebook Marketing: Create and Upload Product Feeds to Facebook Catalogue Manager

Few Words About The Facebook Catalogue Integration:

Create Product feeds and upload products to the Facebook Catalogue Manager, This will put the products to the social platform and attract customers to your online shop. This will increase the revenue.

Work Flow: How The Facebook Catalogue Integration Works

  • Create categories using their corresponding codes
  • Get all Google product categories, Check "https://www.google.com/basepages/producttype/taxonomy-with-ids.en-US.txt"
Product Brand:
  • Create the brand names you use
  • Used with product as additional attribute
Additional Attributes for Product :
  • These attributes are used to identify product uniquely in International Market
  • Use these fields, to map with facebook fields
