Customer Unique Code

Assign Unique Code to Customer for Reference

Few Words About The Customer Unique Code:

Assign a Unique Reference Code to each Customer and use this code to identify each customer uniquely.

Work Flow: How The Customer Unique Code Works
Generate Customer Ref. Code
  • Go to the General Settings
  • Click the button "Generate Customer Code" to generate code for those all customers if they do not have code.(Recommended to use only one time after the installation of this module, this will generate code for all existing customers.)
  • When Create New customers the code will assigned by itself
Customer Code in Form
Customer Code in List
Search Customer
  • Search for the customer using the unique ref. code
Search Results
Identify Customer in POS
  • By using the Ref. Code customer can be identified uniquely
  • Also customer can be searched by ref. code
