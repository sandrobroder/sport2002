Website Scroll Back To Top

This module allows you to go return to the top instantly in the website. If your page is too long then you can go top from the bottom of the page. So when you want to scroll then you can go to the top in a single click.

How it works? click here to watch video

Features

  • You can go to the back to top using one button in the website.
  • No more unnecessary scroll.
  • No matter how long is your page size, you can go to the top quickly.
