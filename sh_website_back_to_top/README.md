About
============
This module allows you to go return to the top instantly in the website. If your page is too long then you can go top from the bottom of the page. So when you want to scroll then you can go to the top in a single click.

Installation
============
1) Copy module files to addon folder.
2) Restart odoo service (sudo service odoo-server restart).
3) Go to your odoo instance and open apps (make sure to activate debug mode).
4) click on update app list. 
5) search module name and hit install button.
