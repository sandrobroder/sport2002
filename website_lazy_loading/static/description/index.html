Website Lazy Loading

Move Past Pagination and scroll-in the products on the same screen with lazy scrolling

Our Lazy Load allows the customers to seamlessly scroll down the products without any interruptions like clicking on the page number to load the next page. Lazy scrolling lets the products to get displayed in continuous rows as the customers keep scrolling downwards.

Note: Website Lazy Loading will only work with the Odoo default theme. To make it work with the customized theme or any third party theme you can contact us.

The customers can see all the products simply by scrolling down the page and add them to cart without the need to jump from page to page.

Lazy loading is also mobile responsive

The module is mobile responsive which means it automatically detect the screen size on which the website is being viewed and adjust the number of products to be displayed in a row.
