Around the world, more than 300,000 e-commerce stores run on PrestaShop technology. The company also received the 2016 CMS Critic Award for Best e-commerce Software.

Hence, it is not wrong to say that Prestashop is one of the strongest E-commerce platforms. However, to run an e-commerce without any hindrance you also need a strong Inventory Management System.

Odoo has a lot for your business management; whether it is inventory management or accounting; Odoo offers you the best solutions for it. Odoo is also becoming an emerging E-commerce platform.

You can also connect to multiple e-commerce and marketplaces to your single Odoo instance with the help of Odoo Multi-channel Sale.

Connect Your Odoo With Multiple Prestashop Instances

If you have one or more website(s) in Prestashop and are still captivated by the strong backend management of Odoo then Odoo Multichannel Prestashop Connector can work as an ideal solution for you.

Odoo Multichannel Prestashop Connector connects the two platforms and lets you use the best of two as per your requirements.

Importing Catalogs in Odoo

You can import Data like Products, Categories, attributes, as well as features from Prestashop to Odoo.

The module makes the Import process quick and easy. You can import data manually or via cron scheduler. The data is stored in the form of feeds in Odoo and then automatically creates data on the basis of the settings.

Mapping is created in the module to avoid duplicity of the data.

Exporting Catalogs in Odoo

Odoo Multichannel Prestashop Connector has made the export process really convenient. Now, you can export products along with its categories from Odoo to Prestashop.

You can export products directly from the specific product form.

Importing Orders in Odoo

You can import Orders along with the data associated with it from Prestashop to Odoo. The import function is simplified with the help of module Odoo Multichannel Prestashop Connector.

Orders statuses like cancel, delivery slip and invoice are automatically imported at the same time.

Moreover, data like customers and their addresses, taxes, delivery methods, payment methods, currency and discount is also imported along with it.

Key Features

Below is the detailed list of features for Odoo Multichannel Prestashop Connector module

Importing Products in Odoo

  • Products are manually imported in bulk
  • You can also import products via cron scheduler
  • Product details (like category, attributes, attribute values) are imported along with the products

Importing Categories in Odoo

  • You can manually import categories in Odoo Or, can also import it via cron scheduler

Importing Customers in Odoo

  • Customers are manually imported in bulk
  • You can also import customers via cron scheduler
  • Customer details (like addresses) are imported along with the Customers

Importing Orders in Odoo

  • Orders are manually imported in bulk
  • You can import orders via cron scheduler
  • Order details (like customers and their addresses, taxes, delivery methods, payment methods, currency) are imported along with the orders
  • Order statuses (like cancel, delivery slip and invoice) are also imported along with the orders

Manually Export Products from Odoo

  • You can export products in bulk Or, export selected products

Manually Export Categories from Odoo

  • You can export categories in bulk Or, export selected categories

Importing Delivery Methods in Odoo

  • You can manually import delivery methods in Odoo Or, can also import it via cron scheduler

Manually Map Orders Statuses

  • You can manually map order statuses from Odoo end
  • Fill the corresponding Order status of Prestashop and Odoo in the Order Status table

Real time Synchronization

  • You can sync order status like shipment (as delivery slip at prestashop end), and Inventory from Odoo to Prestashop in real-time.
  • The real-time order sync from Prestashop to Odoo needs to be added through a separate patch. Avail the free copy from Our Support Team.

Enhanced Dynamic Dashboard

  • Introduced a dynamic dashboard to track the data of Prestashop to your Odoo.
  • Import/Export the data of any channel directly from the dashboard.
  • Well-designed Donut Chart and Line Graph to analyze the data.
  • The dashboard also offers Clickable tabs and data cards.

Managing Synchronization History

  • All the synchronization history is managed in Odoo

Donut Chart to Analyze the data

  • Drop down to Select the Data Type (Order and Product)
  • Track the data of connected Prestashop

Line Graph to Analyze the data

  • Drop down to Select the Data Type (Order, Product, Category and Customer)
  • Track the data the Line Graph

Clickable Data Cards for the Details

  • View the imported Product, Category, Order and Customers in the Tabs
  • The tabs are clickable and direct you to the respective mapping, feeds as well as sync history.
